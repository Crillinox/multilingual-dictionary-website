# Contributing

Thanks for helping improve the dictionary! Contributions are simple—just follow the formats below.

---

## General Rules

* Use `|` to separate fields
* Keep everything on one line per entry
* Use `/` to separate multiple definitions
* Keep part of speech lowercase (e.g. `noun`, `verb`, `adjective`)
* Avoid extra spaces unless necessary

---

## Chinese Format

```
traditional|simplified|pinyin|definition|partOfSpeech
```

### Example:

```
愛|爱|ài|love|verb/noun
貓|猫|māo|cat|noun
```

---

## English / Spanish Format

```
word|definition|partOfSpeech
```

### Example:

```
house|a building for people to live in|noun
run|to move quickly on foot|verb
```

---

## Japanese Format

```
word|definition|partOfSpeech
```

### Notes:

* Use **kanji** when available (e.g. 塩)
* Use **katakana** for loanwords (e.g. チキン)
* Use **hiragana** for native words without kanji (e.g. そして)

### Example:

```
塩|salt|noun
チキン|chicken (loanword)|noun
そして|and then / and|conjunction
```

---

## Tips

* Keep definitions concise and clear
* Avoid duplicate entries
* Double-check spelling and formatting

---

## Submitting Changes

* Add your entries to the appropriate file
* Make sure formatting matches the examples
* Submit a pull request or send the entries directly

Thanks for contributing
