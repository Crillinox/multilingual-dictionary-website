# Update 1.0 Highlights: 

* Added Chinese and Spanish dictionaries
* Improved search system:

  * Exact, prefix, and partial matching
  * Search by characters, pinyin, or definition
  * Tone-insensitive pinyin matching (e.g. "mao" → "māo")
* Added part-of-speech filtering
* Multi-sense definitions displayed as clean lists
* Click a result to instantly view full entry
* Added definition search for English and Japanese
* Added support for community-contributed entries

### UI Improvements

* Updated light and dark mode
* Each dictionary now has its own color theme
